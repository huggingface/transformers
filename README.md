# 🤗 Hugging Face Transformers - Learning Notes & Contributions

This repository documents my learning journey and contributions to the Hugging Face Transformers ecosystem. It includes improvements to documentation, examples, tutorials, and bug fixes while exploring modern Natural Language Processing (NLP) and Large Language Models (LLMs).

## 📖 About

Hugging Face Transformers is one of the most widely used open-source libraries for building applications with transformer-based models such as BERT, GPT, T5, RoBERTa, LLaMA, and many others.

This repository is intended to:

* Learn transformer architectures
* Understand tokenizer and model pipelines
* Explore inference and fine-tuning
* Contribute to open-source documentation and examples
* Practice writing production-quality Python code
