# BEATs

## Overview

BEATs (Audio Pre-Training with Acoustic Tokenizers) was proposed in
[BEATs: Audio Pre-Training with Acoustic Tokenizers](https://arxiv.org/abs/2212.09058)
by Sanyuan Chen, Yu Wu, Chengyi Wang, Shujie Liu, Daniel Tompkins, Zhuo Chen, Furu Wei.

BEATs is an iterative audio pre-training framework that trains audio models to predict
acoustic tokens generated by a discrete tokenizer. The model uses a 2D patch embedding
on Mel Filterbank spectrograms (similar to ViT for images) followed by a Transformer encoder.

The abstract from the paper:

*We propose BEATs, an iterative audio pre-training framework to learn Bidirectional Encoder
representation from Audio Transformers. In the first iteration, we use a randomly initialized
acoustic tokenizer to generate discrete labels, and train an audio SSL model with the masked
label prediction task. After the first iteration, the audio SSL model is used as the acoustic
tokenizer for the next iteration of audio pre-training. We show that BEATs achieve state-of-the-art
results on various audio and sound classification tasks.*

This model was contributed by the HuggingFace team based on the
[original implementation](https://github.com/microsoft/unilm/tree/master/beats) by Microsoft Research.

## Usage Example

```python
import torch
import numpy as np
from transformers import BEATsFeatureExtractor, BEATsModel, BEATsForAudioClassification

# Load feature extractor and model
feature_extractor = BEATsFeatureExtractor()
model = BEATsModel.from_pretrained("microsoft/beats-base")

# Prepare audio input (16kHz)
audio = np.random.randn(16000).astype(np.float32)
inputs = feature_extractor(audio, return_tensors="pt")

# Extract features
with torch.no_grad():
    outputs = model(inputs["input_values"].squeeze(0))

print("Hidden states shape:", outputs.shape)
# (batch, seq_len, 768)
```

For audio classification:

```python
from transformers import BEATsForAudioClassification, BEATsFeatureExtractor
import torch
import numpy as np

feature_extractor = BEATsFeatureExtractor()
model = BEATsForAudioClassification.from_pretrained("microsoft/beats-base")

audio = np.random.randn(16000).astype(np.float32)
inputs = feature_extractor(audio, return_tensors="pt")

with torch.no_grad():
    outputs = model(inputs["input_values"].squeeze(0))

predicted_class = outputs["logits"].argmax(-1)
print("Predicted class:", predicted_class)
```
